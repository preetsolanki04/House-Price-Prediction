# 🏠 House Price Prediction Project

## 📌 Overview

This project focuses on predicting house prices using Machine Learning techniques. It analyzes various features such as location, area, number of bedrooms, and other factors to estimate the price of a house.

The goal of this project is to build an accurate predictive model and understand the factors influencing housing prices.

---

## 🚀 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature selection and engineering
* Model training and evaluation
* Price prediction using trained model

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries Used:**

  * NumPy
  * Pandas
  * Matplotlib
  * Seaborn
  * Scikit-learn

---

## 📊 Workflow

1. Import dataset
2. Data cleaning & preprocessing
3. Perform Exploratory Data Analysis (EDA)
4. Feature engineering
5. Split data into training and testing sets
6. Train Machine Learning models
7. Evaluate model performance
8. Predict house prices

---

## 🤖 Machine Learning Models Used

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

---

## 📈 Model Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

---

## 📌 Results

* The model successfully predicts house prices with good accuracy.
* Feature importance analysis helps understand key factors affecting prices.
